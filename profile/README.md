# Berkeley Humanoids

Berkeley Humanoids is a project initiative focused on building full-stack humanoid robot systems. Our work centers on system-architecture designs that connect robot hardware with modern learning-based policies, and on demonstrating end-to-end capabilities on real robots.

We develop humanoid mechanical hardware, robot-control software, computing systems, teleoperation and data-collection pipelines, simulation assets, and deployment infrastructure as part of a unified research stack. Through this stack, we study how humanoid robots can be built, controlled, trained, and deployed as complete integrated systems rather than as isolated components.
